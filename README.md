## `UltralightUnity`

It is a bride between unity and `ultralight v1.4`  

Currently, it is working on `Linux` & `Windows`  

## Getting started

**Unity Addressables**
- If you don't have Addressables installed, you need to configure it first.
- Go to: Window → Asset Management → Addressables → Groups
- Click **Create Addressables Settings**

**Unity Old Input System**
- Go to: Project Settings → Player → Other Settings → Configuration → Active Input Handling
- Set **Active Input Handling** to `Both` or `Old` depending on your needs.
- If you don't want to use the old input system, edit `ULInputSender` and `ExampleSiteController`.
  *(New input system support may be added in the future. Feel free to contribute at [Unity Sample](https://github.com/latacko/UltralightUnitySample))*


### Installation

1. Download `unitySample.unitypackage` from [Releases](#)
2. Import it into your project
3. Read the `README.md` inside the sample
4. Open the example scene at: `Latacko/Ultralight/Example/Example_Scenes/UltralightSample`

### Required Components

- `UltralightManager`
- `UnityFileSystem`
- `ULView`

### Ready-to-Use Prefabs

- `UltralightManager`
- `WebView` *(fullscreen browser)*
